# 🚀 EduNova — AI Powered Learning Platform

EduNova is a modern full-stack education platform designed to provide students with an interactive, intelligent, and futuristic learning experience.  
The platform combines course management, AI-powered assistance, analytics, and responsive dashboards into one seamless ecosystem.

---

# 🌟 About The Project

EduNova was created to redefine online education through modern technology and premium user experience.

The platform enables students to:
- Access and explore courses
- Track learning progress
- Manage assignments
- Interact with an AI learning assistant
- Monitor analytics and performance
- Learn through a modern and engaging interface

This project follows a scalable full-stack architecture and is built using industry-standard technologies.
