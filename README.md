# EduNova — AI Powered Learning Platform

Premium education platform (React/Vite + Node/Express + MongoDB).

## Requirements

- Node.js 18+
- MongoDB (or MongoDB Atlas)

## Setup

Create environment variables:

```bash
copy .env.example .env
```

## Install

```bash
npm install
```

## Run (Dev)

```bash
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:8080

## Seed demo data

```bash
npm run seed
```

## Build

```bash
npm run build
```

## Notes

- The AI assistant uses mock responses.
- Admin panel is protected by user role.
